# UrbanNest E-commerce Site

This is a responsive e-commerce website for a modern furniture brand called **UrbanNest**.

## Pages
1. **Home (`index.html`)**: Features hero section, categories, and newsletter.
2. **Products (`products.html`)**: A responsive grid of 6 furniture items with hover effects.
3. **About Us (`about.html`)**: Company story and core values.
4. **Contact (`contact.html`)**: Contact information and a functional contact form.
5. **Account (`account.html`)**: Combined Login and Sign-in forms.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Interactive Elements**: Hover effects on all buttons, links, and product cards.
- **Form Validation**: Basic JavaScript validation for all forms.
- **Local Assets**: All images are stored locally in the `img/` folder for reliability and offline access.
- **Modern UI**: Clean, minimalist aesthetic using the Inter font.

## How to Host
To fulfill your assignment's hosting requirement, you can use **Netlify** (highly recommended for its simplicity):
1. Create a free account at [Netlify](https://www.netlify.com/).
2. Drag and drop this folder (`Furniture`) into the Netlify "Sites" dashboard.
3. Your site will be live in seconds!

Alternatively, you can use **GitHub Pages**:
1. Upload these files to a new GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder, then click **Save**.
